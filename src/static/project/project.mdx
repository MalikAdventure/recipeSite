# О проекте

## Pet-проект сайта с рецептами на Next js

### Реализованные функции на сайте:

- Создать и войти в свой аккаунт
- Создать совой рецепт на сайте
- Посчитать свои каллории
- Найти рецепт по поиску и фильтрам
- Посмотреть новости

### Используемые технологии на сайте:

- **Next js** - JavaScript фреймворк, созданный поверх React
- **Redux toolkit** - глобальный state manager
- **MDX** - совместное использование Markdown с JSX
- **Scss** - предпроцессор css
- **TypeScript** - JavaScript с типизацией
